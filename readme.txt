Easily put HTML5, mobile friendly, video on your WordPress website. Integrates directly with Zencoder, or use with already encoded video.

== Description ==

Mobile Video for WordPress is designed to make video easy for WordPress. It integrates with Zencoder and has video training available that shows you how to do each step. There is no requirement for flash to view video, although flowplayer is used as a fallback when that's the only working option.

[HTML5Video](http://mobilevideoforwordpress.com/ "Mobile Video for WordPress")
